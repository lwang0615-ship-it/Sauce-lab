# Sauce-lab

An app for guiding formula development.

## Overview

Sauce-lab is designed to assist in the development and optimization of formulas through a guided, interactive experience.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/lwang0615-ship-it/Sauce-lab.git
cd Sauce-lab
npm install
```

### Running the Application

```bash
npm start
```
